## Code Contributions

### Getting Started

1. **Fork the repo**
```bash
git clone https://github.com/yourusername/gravity-hack.git
cd gravity-hack
```

2. **Create a branch**
```bash
git checkout -b feature/your-amazing-idea
```

3. **Make your changes**
- Write clean, commented code
- Add tests if applicable
- Update documentation

4. **Test your changes**
```bash
python -m pytest tests/
python gravity_modulator.py
```

5. **Submit a PR**
```bash
git push origin feature/your-amazing-idea
```

Then open a Pull Request with:
- Clear description of what you changed
- Why you changed it
- Any relevant physics/engineering justification

## Code Style

- **Python:** Follow PEP 8
- **Comments:** Explain the physics, not just the code
- **Documentation:** Update README if you add features
- **Tests:** Add tests for new functionality
